# Vesta F1 in Schools Website

Welcome to the Vesta F1 in Schools website documentation.

## Overview
The Vesta F1 in Schools website is designed to provide information about the Vesta team's participation in the F1 in Schools program. Our goal is to encourage young minds to engage in STEM fields through engaging racing simulations.

## Features
- **Information about the team:** Learn about our team members, roles, and achievements in the F1 in Schools competition.
- **Interactive chatbot:** We have an AI-powered chatbot that can answer questions about the Vesta team, the competition, and provide guidance on related topics.
- **Live race simulations:** Engage with our real-time racing simulations and experience the thrill of the competition.
- **Resources section:** Access various resources, including tutorials, blogs, and events related to the program.
- **Contact us:** A section dedicated to reaching out for inquiries or collaboration opportunities.

## Chatbot Capabilities
Our integrated chatbot can:
- Answer FAQs about the F1 in Schools program.
- Provide insights on how to join or start a team.
- Offer guidance on design and strategy for F1 racing.
- Assist in navigating our website and finding information quickly.

## Deployment Instructions
To deploy the Vesta F1 in Schools website, follow these steps:
1. Clone the repository:
    ```bash
    git clone https://github.com/a8dya/vesta-f1-website.git
    ```
2. Navigate into the project directory:
    ```bash
    cd vesta-f1-website
    ```
3. Install the required dependencies:
    ```bash
    npm install
    ```
4. Launch the application:
    ```bash
    npm start
    ```
5. Open your browser and visit `localhost:3000` to view the website.

For further information, refer to the [documentation](docs/README.md) or contact our support team.
